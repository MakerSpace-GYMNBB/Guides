# Satellite Image Guide
## Description
The purpose of this guide is to teach how to receive images from weather satellites for example the NOAA15-20 satellites. The guide is meant for use in education for example geography class. It will teach you how to find a satellite, get the coressponding raw data from it and decode this data into an actual image.

## Requirements
PC with Windows / MacOS / Linux
Internet connection to get the raw audio data from the satellite.
